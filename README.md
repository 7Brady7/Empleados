Sistema de Gestión de Empleados (CRUD en C# WinForms)

Este proyecto es una aplicación de escritorio desarrollada en C# con Windows Forms, que permite realizar operaciones básicas de gestión de empleados:

Agregar empleados

Editar información

Eliminar empleados

Consultar registros

Limpiar campos del formulario

Calcular descuentos de ARS y AFP automáticamente
Tecnologías utilizadas

C#

.NET Framework / .NET (especifica tu versión)

Windows Forms

DataGridView
